To compile this program in this directory (source), make a directory called bin.
Navigate to the ./source/src directory and run the following command

javac *.java */*.java */*/*.java -d ../bin

Navigate back to the parent folder ./source and you can run the program with the following command

java -cp bin Main

Alternatively you may execute the jar file to run the program by double clicking the executable jar CaraLoft.jar or using the command java -jar CaraLoft.jar in the ./source folder.
